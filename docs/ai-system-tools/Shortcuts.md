## Applications
| Shortcut | Action |
|---|---|
| `Super + Enter` | Open terminal |
| `Super + B` | Open browser |
| `Super + E` | Open file manager |

## Windows
| Shortcut | Action |
|---|---|
| `Super + Q` | Close window |
| `Super + Shift + Q` | Kill process entirely |
| `Super + F` | Fullscreen |
| `Super + M` | Maximize |
| `Super + W` | Toggle floating |
| `Super + J` | Toggle split |
| `Super + K` | Swap split |
| `Super + mouse drag (left)` | Move window |
| `Super + mouse drag (right)` | Resize window |

## Actions
| Shortcut | Action |
|---|---|
| `Super + Shift + S` | Screenshot — freeze screen, select region, copy (flameshot) |
| `Super + Tab` | Window overview |
| `Super + Ctrl + W` | Wallpaper selector |
| `Super + R` | App launcher |
| `Super + Shift + B` | Reload waybar |
| `Super + Shift + R` | Reload Hyprland config |
| `Super + Shift + A` | Toggle animations |
| `Super + V` | Clipboard history |
| `Super + S` | Toggle dashboard |

## Workspaces
| Shortcut | Action |
|---|---|
| `Super + 1-9, 0` | Switch to workspace 1-10 |
| `Super + Shift + 1-9, 0` | Move window to workspace 1-10 |
| `Super + N` | Jump to next empty workspace |

## Fn / Media Keys
| Shortcut | Action |
|---|---|
| `Brightness Up/Down` | Screen brightness |
| `Volume Up/Down` | Volume |
| `Mute` | Toggle mute |
| `Mic Mute` | Toggle microphone |
| `Play/Pause/Next/Prev` | Media controls |
| `Calculator key` | Open calculator |
| `Lock key` | Lock screen |
| `Kbd backlight Up/Down` | Keyboard backlight |
| `Ctrl + Alt + Delete` | Open Mission Center (task manager) |
