## Applications
| Shortcut | Action |
|---|---|
| `Super + Enter` | Open terminal |
| `Super + B` | Open browser |
| `Super + E` | Open file manager |
| `Super + Ctrl + E` | Emoji picker |

## Windows
| Shortcut | Action |
|---|---|
| `Super + Q` | Close window |
| `Super + Shift + Q` | Kill process entirely |
| `Super + F` | Fullscreen |
| `Super + M` | Maximize |
| `Super + T` | Toggle floating |
| `Super + Shift + T` | Float all windows on workspace |
| `Super + Alt + T` | Float + pin window |
| `Super + J` | Toggle split |
| `Super + K` | Swap split |
| `Super + G` | Toggle window group |
| `Super + ↑ ↓` | Move focus up/down |
| `Super + Shift + ← → ↑ ↓` | Resize window |
| `Super + Alt + ← → ↑ ↓` | Swap windows |
| `Super + mouse drag (left)` | Move window |
| `Super + mouse drag (right)` | Resize window |
| `Alt + Tab` | Cycle windows |

## Workspaces
| Shortcut | Action |
|---|---|
| `Super + 1-9, 0` | Switch to workspace 1-10 |
| `Super + Shift + 1-9, 0` | Move window to workspace 1-10 |
| `Super + Ctrl + 1-9, 0` | Move ALL windows to workspace 1-10 |
| `Super + N` | Jump to next empty workspace |
| `Super + →` | Next workspace |
| `Super + ←` | Previous workspace |
| `Super + scroll` | Scroll through workspaces |
