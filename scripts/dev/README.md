# Dev Tools

Development runtimes, editors, and CLI tools.

## Usage

```bash
bash scripts/dev/install.sh
```

## What gets installed

| Tool | Description |
|------|-------------|
| Neovim | Terminal text editor |
| Zed | Code editor |
| Node.js + npm | JavaScript runtime |
| Rustup | Rust toolchain manager (installs stable) |
| Python + pip + pipx | Python runtime and package managers |
| Docker + Compose | Containers |
| GitHub CLI | `gh` — interact with GitHub from terminal |
