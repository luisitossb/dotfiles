# Apps

User-facing applications. Run this on any machine after the main rice is set up.

## Usage

```bash
bash scripts/apps/install.sh
```

## What gets installed

| App | Description |
|-----|-------------|
| Mission Center | GUI system monitor |
| btop++ | Terminal resource monitor |
| nvtop | GPU process monitor |
| Zen Browser | Firefox-based browser |
| Discord | Chat |
| Telegram | Messaging |
| Spotify | Music |
| VLC | Media player |
| Obsidian | Markdown notes |
| qBittorrent | Torrent client |
| Open Video Downloader | GUI for yt-dlp |
| Steam | Gaming platform |
| Proton-GE | Improved Steam compatibility layer |
| Moonlight | Game stream client (connects to Sunshine) |
| Wine + Winetricks | Windows app compatibility |
| LocalSend | Local file sharing |
